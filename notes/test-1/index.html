Test 1

Test 1, at class time on October 12, may include the following tasks:

  1. Define “logically valid.”
  2. Answer true-false questions using the terms/concepts from the vocabulary sheet, very similar to the true-false exercises you’ve done.
  3. Look at a few sentences, and construct a symbolization key for the atomic propositions in those sentences—one adequate to bring out their truth-functional structure.
  4. Using a symbolization key, symbolize English sentences in TFL.
  5. Using a symbolization key, translate symbolized TFL sentences into English.
  6. Construct truth tables for symbolized sentences to show, variously:
    • whether a sentence is a tautology, a contradiction, or contingent;
    • whether a set of sentences is jointly possible/consistent or not;
    • whether an argument is logically valid or not.
  7. For a constructed table, write a sentence saying why/how it shows the answer to the question.
