Topic 3: Probability

Here we set up some basic ideas about probability which will be key for understanding the approaches to scientific justification which have dominated the last several decades.

Reading

  1. Hacking, Introduction to Probability and Inductive Logic
    • Chapter 4 “Elementary Probability Ideas”
    • Chapter 5 “Conditional Probability”
    • Chapter 11 “What Do You Mean?”
